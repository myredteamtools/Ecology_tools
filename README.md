# Usage
1. Set your php or other enviroments to config.properties
2. Get phpggc
``` bash
cd libs
git clone https://github.com/ambionics/phpggc 
```
3. Install requirements
``` bash
python3 -m pip install -r requirements.txt
```
4. Run?
``` bash
python3 main.py
```
5. Have fun.
